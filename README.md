How to run project's backend on your device?

1. Download this repository
2. Open the extracted folder with your code editor (vscode)
3. Switch to server folder: cd ./server
4. Run: "npm install" on terminal to install npm packages that are used through project
5. Run: "npm run create" on vscode terminal (to create database on your device)
6. Run: "npm run up" to create tables on your database
7. Run: "npm run dev" on terminal to run the backend server on your device's localhost
